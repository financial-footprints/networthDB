# NetworthDB

The scope of this project is to read financial institution statements and convert data to SQL format.

## Setup

### Rust Installation

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
# Verify
rustc --version
```

### Setup Repository

```sh
cargo build
cargo test
cargo run
```
